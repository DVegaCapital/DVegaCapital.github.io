Portfolio optimization using R lpsolveAPI package (Linear programming)

Posted on 2016-12-29

Linear programing is a valuable instrument when it comes to decision making. It is a method to achieve the best outcome(such as maximum profit or lowest cost) in a mathematical model whose requirements are represented by linear relationship. Here I will present how R in conjunction with lpsolveAPI package, can be used to build a linear programming model to solve portfolio construction and optimization issue we met at work.

Portfolio construction analysis

We want to build 10 portfolio with 7000 trades in our bucket. Each trades has its own delta value and each portfolio has its own target delta range (delta_min and delta_max). We are looking for a way to maximize the number of trades in these portfolios.

The following constrains have to be taken in consideration:

  • Each portfolio delta range (delta max and delta min)
  • Each trade can only be put in one portfolio
